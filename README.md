# KORMAP-TV

웨이브와 티빙의 실시간 채널을 Android TV와 Google TV 환경에서 편하게 시청할 수 있도록 정리한 라이브 TV 앱입니다.

기존의 유사 프로그램이 더 이상 정상적으로 동작하지 않는 상황에서, 개인적인 시청 목적에 맞춰 정리해 사용하던 앱을 같은 불편을 겪는 분들에게 도움이 될 수 있도록 공개한 프로젝트입니다.

## 공지

- 2026-04-08 기준 TVING 로그인이 현재 정상적으로 진행되지 않는 사례가 확인되고 있습니다.
- 해당 이슈는 확인 중이며, `v26` 버전으로 패치 작업을 진행하고 있습니다.
- 진행 상황은 이 저장소의 README 또는 이슈를 통해 추가로 안내하겠습니다.

## 주요 기능

- WAVVE, TVING 실시간 채널 시청 지원
- TVING `AUTO` 화질 지원
  - 가능한 최고 화질부터 시도하고, 재생이 막히면 하위 화질로 자동 전환
- TVING 시청 가능 채널 중심 노출
  - 재생이 막히는 채널은 목록에서 최대한 정리
- TVING 채널 상태 배지 표시
  - 채널별 최고 확인 화질 기준으로 `FHD`, `HD`, `SD`, `M` 표시
- WAVVE 로그인 정보 저장 시 즉시 검증
  - 잘못된 계정 정보는 설정 단계에서 바로 확인 가능
- 시청 중 `확인` 키 길게 누르기 지원
  - 즐겨찾기, 화질 관련 설정 등 자주 쓰는 기능에 빠르게 접근 가능
- 로그인 정보, 화질 설정, 서비스 설정 유지
  - 로그아웃 전까지 저장된 값이 다음 실행에도 이어짐
- Android TV, Google TV 리모컨 환경에 맞춘 사용성 구성

## 유의사항

- WAVVE와 TVING의 실시간 채널을 시청할 수 있는 적절한 유료 상품이 필요합니다.
- 이 앱은 WAVVE, TVING과 직접적인 관련이 없는 비공식 앱입니다.
- 서비스 정책, 방송사 판권, 계정 권한, 디바이스 조건에 따라 일부 채널은 재생이 제한될 수 있습니다.
- 서비스 제공 방식이나 정책 변경, 외부 환경 문제 발생 시 일부 기능 또는 해당 서비스 지원이 예고 없이 중지될 수 있습니다.
- 일부 채널은 제공 화질이 다르므로 설정 화질과 실제 재생 화질이 다를 수 있습니다.

## 안내사항

- 로그인 정보와 화질 설정은 사용자 기기 내에 저장되며, 로그아웃 전까지 다음 실행에도 유지됩니다.
- 로그인 정보는 개발자 운영 서버로 별도 전송하거나 수집하지 않으며, 각 서비스 로그인과 재생 처리에 필요한 범위에서만 사용됩니다.
- TVING은 웹 로그인 기반 세션을 사용하므로, 최초 로그인 이후 저장된 세션을 재사용합니다.

## 개인정보 및 통신 안내

- 이 앱은 사용자 계정 정보, 시청 이력, 사용량 통계, 행동 분석 데이터를 개발자 서버로 전송하지 않습니다.
- Firebase용 Google Analytics를 사용하지 않으며, 사용량 분석이나 통계 수집을 위해 Google Firebase 등 외부 분석 플랫폼으로 데이터를 전송하지 않습니다.
- 앱 내부 설정값과 로그인 유지 정보는 사용자 기기 내에 저장되며, 개발자가 별도로 수집하지 않습니다.
- 단, WAVVE와 TVING 로그인 및 실시간 재생을 위해 각 서비스의 공식 서버와 통신은 반드시 필요합니다.
